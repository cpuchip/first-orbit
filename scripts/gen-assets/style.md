# First Orbit — art direction (the shared style preamble)

Every generated asset is prefixed with this so the set stays visually coherent.

**Universe:** The Hubble Frontier, *origin era* — humanity's first reach to orbit,
the decades before faster-than-light travel, when the Mars Orbital Shipyards
("Mars-field") are first being laid. Grounded near-future spaceflight, not space
opera. Think Apollo-meets-2030s: real rivets, real plumbing, real heat shielding.

**Look:**
- Clean retro-futurist 2D game art. Flat-but-detailed, crisp edges, readable at
  small sizes.
- Muted industrial palette — brushed aluminium, off-white, slate grey — with a
  restrained cyan-and-amber accent (the MARS interface colours from the IP bible).
- Lighting is soft and consistent: key light from the upper-left, gentle ambient
  fill. No dramatic rim-light unless asked.
- Hardware reads as *engineered*: panel lines, fuel lines, nozzle bells, decoupler
  rings. Nothing chrome-shiny or cartoonish.

**Technical:**
- Part icons and props: centred subject, **transparent background**, side-view
  (the game is 2D side-on), nose pointing up.
- Planets/moons: top-down circular disk, subtle texture, soft terminator.
- No text or lettering baked into art unless the asset IS a wordmark.
- No real-world franchise iconography (this is original IP).
